# Client (React + Vite)

The frontend for the Student Support System.

For the full architecture, component reference, and how everything fits together, see the [root README](../README.md).

## Quick start

```bash
npm install
npm run dev      # http://localhost:5173 (proxies /api to the backend on :5000)
```

| Script            | Does                         |
| ----------------- | ---------------------------- |
| `npm run dev`     | Start the Vite dev server    |
| `npm run build`   | Production build             |
| `npm run preview` | Preview the production build |
| `npm run lint`    | Run ESLint                   |
